# Task-Management-System
Java/Spring Boot/Spring Security

Task Management System – это веб-приложение для создания, назначения, отслеживания и управления задачами. Оно поддерживает ролевой доступ (Админ, Менеджер, Пользователь) и обеспечивает структурированный рабочий процесс для команд.

Ключевые функции
✅ Управление пользователями – Безопасный вход и ролевой доступ.
✅ Работа с задачами – Создание, назначение, обновление и удаление задач со статусами.
✅ Панель управления – Фильтрация, сортировка и распределение задач.
✅ Уведомления – Email/системные оповещения об изменениях и дедлайнах.
✅ База данных – Безопасное хранение данных и история задач.
✅ API – RESTful-интерфейсы для интеграции с внешними сервисами.

Технологии
🔹 Backend: Spring Boot (Java) + Spring Security
🔹 Frontend: Thymeleaf (HTML, CSS, JS)
🔹 База данных: MySQL
